# git-basics-lab
Git basics lab exercises for ParoCyber DevSecOps training.

**Author:** Abigail

## What I practiced
- Setting up Git identity (`user.name`, `user.email`) and understanding global vs. local config
- Initializing a repository and exploring the `.git` folder structure
- Staging and committing changes, and understanding untracked vs. modified vs. staged files
- Viewing history with `git log`, `git log --oneline`, `git log --stat`, and `git log --graph`
- Inspecting individual commits with `git show`
- Comparing changes across commits with `git diff`
- Connecting a local repository to GitHub with `git remote add`
- Resolving a real merge conflict between local and remote histories using `git pull --allow-unrelated-historie
